# 変更履歴

マニュアルのバージョン履歴。

## バージョン 1.1.0（2026年7月15日）

**改訂版論文との整合**

### 変更
- 研究の位置づけを、絶対計測ではなく**相対的な生育モニタリング／変化検出**として再定義（研究内容・形質抽出ページ全体、主要指標はCV）。
- 投稿先を *Computers and Electronics in Agriculture*（Elsevier）に修正、状況は**査読中**（旧記載の IEEE TCE／「準備中」を訂正）。
- 連絡先メールを `zobaer.al.24@shizuoka.ac.jp` に修正。

### 追加
- **生物学的検証：** 剪定イベント検出（記録された3件、p = 0.0008）。
- **物理基準検証：** 場内の45cmパイプによる裏付け（r = 0.74, p < 0.001）。絶対精度の主張ではなく、実スケールとの整合の裏付けとして提示。
- **限界と今後のグラウンドトゥルース取得手順**（メジャー／剛体較正バー／ArUcoマーカー）。
