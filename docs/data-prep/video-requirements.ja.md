## 撮影プロトコル

### Step 1：カメラ位置

カメラは**固定**した状態で、植物を囲むようにゆっくりと周回しながら撮影します。

!!! info "条件1 vs 条件2"
    本データセットでは**条件1**（単一固定視点）を使用。複数視点を検証した結果、条件1がPSNR 23.80 dBで最高性能を達成しました。

```
      [カメラ]
          |
    約1.2m
          |
      🌱 植物
```

### Step 2：照明条件

| 条件 | 状態 | 理由 |
|-----------|--------|-----|
| 撮影時刻：12:00〜13:00 JST | ✅ 必須 | 一定の自然光、影が少ない |
| 曇天 | ✅ 推奨 | 拡散光 → 鏡面反射が少ない |
| 直射日光 | ⚠️ 避ける | 極端な影がCOLMAPのマッチングを妨げる |
| 人工照明のみ | ⚠️ 避ける | 日付間での色温度のズレが生じる |

## 良い動画・悪い動画の例

=== "✅ 良いフレーム"
    - 葉の縁がシャープ（モーションブラーなし）
    - 植物がフレーム内に完全に収まっている
    - 均一な照明、露出オーバーなし
    - 一定の背景（人が映り込まない）

=== "❌ 悪いフレーム"
    - モーションブラー（手持ち撮影など）
    - 植物がフレームからはみ出ている
    - 露出オーバー／アンダー
    - 前景に遮蔽物がある

## 撮影前チェックリスト

- [ ] カメラのバッテリー残量 50%以上
- [ ] ストレージ空き容量 5GB以上
- [ ] 三脚：完全に固定、ぐらつきなし
- [ ] 時刻：12:00〜13:00 JST
- [ ] 植物：フレーム内に完全収納
- [ ] フォーカス：植物でロック（オートフォーカス無効化）
- [ ] 手ぶれ補正：OFF
- [ ] 撮影時間：最低60秒間連続撮影

## 環境データの記録（研究上重要）

| 変数 | 計測方法 | 重要な理由 |
|----------|---------------|----------------|
| 気温（℃） | 温室センサー | PSNRと相関あり |
| 湿度（%） | 温室センサー | **PSNRとの最強の相関（r = -0.68）** |
| 日射量（W/m²） | 日射計 | 100 W/m²超 → PSNR低下 |
| CO₂（ppm） | CO₂センサー | 副次的な成長因子 |
