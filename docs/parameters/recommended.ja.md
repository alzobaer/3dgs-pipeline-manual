# 推奨設定

検証済みの設定（PSNR：23.71 dB）。

---

## 完全なパイプライン設定

| ステージ | パラメータ | 値 |
|--------|---------|--|
| 動画 | 入力条件 | 条件1（固定視点） |
| フレーム | FPS | 5 fps |
| フレーム | 品質 | qscale:v 2 |
| COLMAP | カメラモデル | single_camera=1 |
| COLMAP | マッチャー | exhaustive |
| COLMAP | BAイテレーション | 20（最大） |
| 3DGS | イテレーション | 30,000 |
| 3DGS | 解像度 | フル4K |

完全な設定は[データセット詳細](../data-prep/dataset.md)を参照してください。
