## 主要な知見

固定視点（条件1、5 fps）が全ての撮影条件を上回る：

| 条件 | PSNR |
|-----|-----|
| **条件1 — 端部a固定、低視点** | **23.71 dB** |
| 条件5 — 一方向a→d、階層1 | 21.42 dB |
| 条件2 — 端部b固定、低視点 | 20.11 dB |
| 条件3 — 往復、階層1 | 19.37 dB |
| 統合（全6条件） | 18.60 dB |

**教訓：** 撮影の一貫性 > 手法の高度さ
